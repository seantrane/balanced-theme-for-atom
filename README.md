## Install <a id="install"></a>

1. Open **Atom** application
2. Go to **Preferences**
3. Go to **Install**
4. Search for `balanced theme`
5. **Install** the _Balanced Theme_ package

Or, install from the command-line:

```bash
apm install balanced-theme
```
